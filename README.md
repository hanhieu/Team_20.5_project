# XanhSM AI Support Chatbot

**Nhóm 29 — Lớp E403 — AI Product Hackathon Day 06**

Chatbot AI hỗ trợ khách hàng và tài xế Xanh SM, phân loại theo vai trò (Hành khách, Tài xế Taxi, Tài xế Bike, Nhà hàng). Dùng RAG + GPT-4o để trả lời từ knowledge base chính thức và dữ liệu cộng đồng Facebook. Có tool calling tra cứu giá cước thực tế theo thành phố.
